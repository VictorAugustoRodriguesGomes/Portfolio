Projeto Hamster

Com objetivo de demonstrar as minhas habilidades como desenvolvedor, desenvolvi esse projeto para gerar um acesso rápido, através de um link ou QR Code, de um contato do WhatsApp, facilitando assim o compartilhamento do mesmo. Para isso, basta informar o número de WhatsApp que deseja compartilhar, e uma mensagem que deseja vincular ou contato (a mensagem sendo opcional). O projeto foi desenvolvido utilizando HTML, CSS, JS (Java Script), Jquery e qrCodeJs.

Projeto Onça-Pintada

Com objetivo de demostrar as minhas habilidades como desenvolvedor, desenvolvi esse projeto que converte valores entre moedas. Para isso, basta informar o valor que pretende converter e selecionar a moeda de origem e a moeda que pretende ser convertida. As moedas utilizadas são: Dólar, Euros e Reais, no futuro pretendo adicionar mais moedas. O projeto foi desenvolvido utilizando a API Awesome juntamente com HTML, CSS e JS (Java Script).

Ark 5

Ark 5 é um projeto de gerenciamento responsável por gerenciar a fila de um restaurante, através do projeto o usuário poderá adicionar pessoas a fila, visualizar as pessoas presentes a fila, trocar a ordem das pessoas presentes na fila, retirar as pessoas que estão na fila e também estar chamando a próxima pessoa presente na fila do restaurante. Os projetos nomeados ARK são uma aplicação desenvolvida em Java na IDE NetBeans. O objetivo é desenvolver um projeto partindo de listas de exercícios encontrados na Internet, seriam projetos mais simples.

Ark 2

Ark 2 é um projeto de gerenciamento de uma biblioteca, através do projeto, usuário poderá efetuar o cadastramento de um livro possuindo 2 parâmetros, sendo eles, o título do livro e o número de páginas, após o cadastro ser realizado o livro é listado utilizando uma JTable. Para realizar essa aplicação, estou utilizando o paradigma da POO (Programação Orientada a Objeto) na linguagem de programação Java. Os projetos nomeados ARK são uma aplicação desenvolvida em Java na IDE NetBeans. O objetivo é desenvolver um projeto partindo de listas de exercícios encontrados na Internet, seriam projetos mais simples.

Ark 1

Ark 1 é um projeto que de maneira aleatória escolhe um número de 0 a 1000 e através de palpites o usuário deverá adivinhar o número escolhido, dessa forma quando mais próximo o usuário estiver do número escolhido a aplicação responderá com a cor verde e quando ele estiver se afastando do número escolhido o projeto responder com a cor vermelha, caso no 13° palpite o usuário não acerte o número escolhido a aplicação informará o número através de um System.out.print Os projetos nomeados ARK são uma aplicação desenvolvida em Java na IDE NetBeans. O objetivo é desenvolver um projeto partindo de listas de exercícios encontrados na Internet, seriam projetos mais simples.
