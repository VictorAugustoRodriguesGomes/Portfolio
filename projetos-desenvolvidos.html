Projeto Sytis

Com objetivo de demonstrar conhecimento adquirido no bacharelado de ciência da computação na Universidade Paulista (UNIP), a atividade prática superlativa número 8 se refere a concepção de duas aplicações em diferentes linguagens de programação que utiliza o mesmo banco de dados capaz de reportar situações em que possam prejudicar o meio ambiente.
Baseado nessas informações desenvolve um site e um aplicativo Android onde por ambas aplicações é possível gerenciar sua conta de usuário(criar uma conta de usuário, realizar o seu login, recuperar a senha através do e-mail e visualizar as informações de sua conta). A partir disso é possível reportar situações em que possam prejudicar o meio ambiente, citando alguns exemplos: pontos de alagamentos, pontos de incêndio, pontos de desmoronamento e assim por diante, também é possível visualizar e comentar as ocorrências adicionadas. O projeto foi desenvolvido utilizando o Android Studio com a linguagem Java e o front-end do site utilizando o HTML, CSS e JS (JavaScript). Para o Back-End utilizei Google firebase.

Projeto Ark 1

Ark 1 é um projeto que de maneira aleatória escolhe um número de 0 a 1000 e através de palpites o usuário deverá adivinhar o número escolhido, dessa forma quando mais próximo o usuário estiver do número escolhido a aplicação responderá com a cor verde e quando ele estiver se afastando do número escolhido o projeto responder com a cor vermelha, caso no 13° palpite o usuário não acerte o número escolhido a aplicação informará o número por um System.out.print. Os projetos nomeados ARK são uma aplicação desenvolvida em Java na IDE NetBeans. O objetivo é desenvolver um projeto partindo de listas de exercícios encontrados na Internet, seriam projetos mais simples.
